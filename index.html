Project Overview

QuantFlow v1.0

An end-to-end algorithmic trading and quantitative risk analytics platform built in Java Spring Boot 3.

Implements production-grade financial algorithms used by quantitative analysts at investment banks, covering the full strategy lifecycle from signal generation to risk reporting.

● 4 trading strategy algorithms

● Full event-driven backtesting engine

● 15+ risk metrics (VaR, CVaR, Sharpe, Sortino...)

● Monte Carlo GBM simulation (10k paths)

● WebSocket live market data feed

● 25+ unit tests with JUnit 5 + AssertJ
